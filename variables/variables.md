# how to create a variable
## create a variable
```go
var a int = 10
var b string = "hello"
var c bool = true;

a := 10
b := "hello"
c := false
```
## create variables
+ 声明多个相同类型的数据
```go
var a, b, c int = 10, 20, 30
a, b, c = 10, 20, 30
```

+ 声明多个不同类型的数据

```go
a, b, c = 10, "hello", true
```
